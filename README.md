# News Scraper

### Rundown

Get news articles from KSL news site and displays them for you to chose which ones you would like to save and read later, also comment on articles.

### Technologies Used

- HTML
- CSS
- Bootstrap
- Node
- Express
- Handlebars
- Axios
- Cheerio
- Mongoose
- MongoDB
